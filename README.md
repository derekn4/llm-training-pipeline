# llm-training-pipeline
quick description under 350 characters  End-to-end LLM training pipeline: pretrains a small GPT from scratch, fine-tunes Llama 3.2 with QLoRA on domain-specific instruction data, aligns it via DPO with a custom preference dataset, and evaluates all checkpoints with lm-evaluation-harness and a validated LLM-as-judge suite.
